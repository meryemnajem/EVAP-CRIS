# EVAP-CRIS
## Simulation d’évaporation triple effet et cristallisation sucrière

Projet académique de Génie des Procédés visant à :
- simuler un évaporateur à triple effet
- modéliser la cristallisation du saccharose
- analyser les bilans thermiques et matière
- évaluer les performances technico-économiques

## Technologies
- Python 3.14
- Flask
- NumPy
- Matplotlib
- CoolProp

python -m venv .venv
